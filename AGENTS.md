# Workspace Agent Rules

## Naming Rule
- Do not create files or folders with spaces in their names.
- Use kebab-case for new file and folder names unless the user explicitly requests a different format.
- If a name would normally contain spaces, replace them with hyphens.

## Scope
- This rule applies to all new files and directories created in this workspace.
- Existing files or folders that already contain spaces should not be renamed unless the user explicitly asks for it.
