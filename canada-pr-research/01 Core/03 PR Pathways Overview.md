# PR Pathways Overview

## Purpose
This note tracks the main Canada PR pathways worth researching for this profile. It is not a legal conclusion and should not contain invented eligibility details.

## Working Rule
- Record only verified facts or clearly mark items as `To Research`
- Use the detailed pathway notes for ongoing updates
