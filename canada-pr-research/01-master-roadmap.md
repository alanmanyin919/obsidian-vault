# Canada PR Master Roadmap

## Purpose
This roadmap breaks Canada PR planning and Canada job application preparation into small, reviewable steps.

## Current Restart Status
- Restart date: 2026-04-04
- Working instruction:
  - Redo the project from Phase 1
  - Treat Phase 3 and later research as draft material until Phase 1 and Phase 2 are re-confirmed
- Immediate focus:
  - Finish Phase 1 before doing new province ranking, pathway ranking, resume execution, or job-search execution

## How To Use This Roadmap
- Work from top to bottom unless priorities change.
- Each item should be split into the smallest practical approval unit.
- Do not mark an item complete until the user has reviewed and approved the result.

## Phase 1: Baseline And Constraints
- [ ] Confirm personal baseline facts
- [ ] Confirm OWP timing, entry window, and practical constraints
- [ ] Confirm financial runway assumptions
- [ ] Confirm mental health and sustainability guardrails

## Phase 2: Migration Reality Check
- [ ] Define what a successful Canada move would look like
- [ ] Define what failure or stop-loss conditions would look like
- [ ] Compare trial move, job-first move, and committed relocation paths
- [ ] Decide whether Canada remains an active plan

## Phase 3: Province And Pathway Research
- [ ] Build first-pass province comparison
- [ ] Research Alberta options
- [ ] Research Manitoba options
- [ ] Research Saskatchewan options
- [ ] Research Atlantic options
- [ ] Review Ontario and British Columbia as higher-cost or higher-competition cases
- [ ] Narrow to a shortlist of realistic provinces
- [ ] Map province shortlist to realistic PR pathways

## Phase 4: Job Path Strategy
- [ ] Rank target role families by realism
- [ ] Review application-support path
- [ ] Review IT-support path
- [ ] Review QA-tester path
- [ ] Review junior-developer path
- [ ] Review operations-support path
- [ ] Choose primary and secondary application tracks

## Phase 5: Resume And Application Assets
- [ ] Finalize base Canada resume
- [ ] Finalize application-support resume version
- [ ] Finalize IT-support resume version
- [ ] Finalize QA-tester resume version
- [ ] Finalize software-developer resume version
- [ ] Create reusable cover-letter structure
- [ ] Create keyword bank from target job postings

## Phase 6: Job Search Execution
- [ ] Build job application tracker
- [ ] Define job search cadence
- [ ] Define tailoring workflow per application
- [ ] Start initial application batch
- [ ] Review response quality and revise documents

## Phase 7: Decision And Execution Readiness
- [ ] Review final shortlist of provinces and role tracks
- [ ] Review financial and mental health readiness
- [ ] Decide whether to proceed with a Canada move
- [ ] If yes, define first 30-day execution plan

## Smallest Task Examples
- Verify one roadmap assumption
- Improve one resume section
- Compare one province on one criterion
- Review one PR pathway eligibility question
- Extract keywords from one job posting
- Revise one role-specific resume summary

## Restart Rule
- If a note from Phase 3 onward conflicts with the rechecked Phase 1 or Phase 2 baseline, the earlier research should be treated as provisional and revised later
