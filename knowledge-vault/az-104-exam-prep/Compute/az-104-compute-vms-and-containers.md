# AZ-104 Compute, VMs, and Containers

## Core Concepts
- Azure Virtual Machines, scale sets, disks, images, and availability features
- App hosting options and managed compute tradeoffs
- Basic container administration in Azure environments

## Know How To Do
- Provision a VM with the right size, OS image, disk, and network settings
- Configure VM extensions and manage start, stop, resize, and redeploy actions
- Understand availability sets, zones, and scaling patterns
- Capture or reuse images for repeatable deployments

## Exam Traps
- Choosing the wrong availability model
- Forgetting managed disk implications
- Mixing up scaling concepts across VM scale sets and other compute services
- Overlooking NSG or route issues when troubleshooting a VM

## Practice Tasks
- Deploy a Linux VM and a Windows VM
- Resize a VM and attach an additional disk
- Configure boot diagnostics and review extension status
- Compare availability set and availability zone deployment patterns
