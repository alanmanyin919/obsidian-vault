# AZ-104 Lab Checklist

Use this as a practical completion tracker.

## Identity and Governance
- [ ] Create users and groups
- [ ] Assign RBAC roles at different scopes
- [ ] Add tags and resource locks
- [ ] Create and review an Azure Policy assignment

## Storage
- [ ] Create a storage account
- [ ] Configure blob and file storage
- [ ] Restrict access with RBAC or networking controls
- [ ] Review redundancy and protection settings

## Compute
- [ ] Deploy a VM
- [ ] Resize a VM
- [ ] Attach and review managed disks
- [ ] Review availability options

## Networking
- [ ] Build a VNet with multiple subnets
- [ ] Apply NSGs
- [ ] Configure VNet peering
- [ ] Validate DNS behavior

## Monitoring and Recovery
- [ ] Create alerts
- [ ] Send logs to Log Analytics
- [ ] Review diagnostic settings
- [ ] Configure and verify backup settings
