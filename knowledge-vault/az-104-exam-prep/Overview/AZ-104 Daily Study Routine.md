# AZ-104 Daily Study Routine

## Objective
Use a small but repeatable routine that fits an office workday.

## Monday to Friday
### Minimum version
- `15-20 min` at lunch or commute:
  review one concept and answer 3 recall questions
- `60-75 min` in the evening:
  one topic only

### Strong version
- `15-20 min` midday recall
- `75-90 min` in the evening
- `10 min` before sleep:
  rewrite key facts from memory

## Saturday
- One `90-minute` block for labs
- One `90-minute` block for theory and recall
- One `30-45 minute` block for weak-topic review

## Sunday
- One `90-minute` mixed review block
- One `60-minute` planning block for the next week
- One `30-minute` light review of flash points and mistakes

## Session Method
1. Pick one outcome only.
   Example: "Understand NSG rules and subnet vs NIC application."
2. Read the note for 15-20 minutes.
3. Close the note and write what you remember.
4. Do one practical action or one scenario question.
5. Write 5 quick prompts for future recall.
6. Mark the topic as `Strong`, `Okay`, or `Weak`.

## Anti-Pattern
- reading for 90 minutes without writing
- switching topics halfway through
- watching videos passively while tired
- collecting resources instead of learning from one note deeply

## Weekly Review Questions
- What topic still feels vague?
- What scenario type slows me down?
- Which mistakes repeat?
- Which note needs a clearer summary?
