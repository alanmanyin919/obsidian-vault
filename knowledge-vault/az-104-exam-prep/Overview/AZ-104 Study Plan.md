# AZ-104 Study Plan

## Goal
Build practical Azure Administrator capability, not just memorized definitions.

## 4-Week Plan
### Week 1
- Identity and access management
- Subscriptions, resource groups, tags, locks, and Policy
- Practice with RBAC assignments and Entra ID user/group administration

### Week 2
- Storage accounts, blob tiers, file shares, replication, and data protection
- Virtual machines, availability options, disks, images, and extensions
- Practice creating and securing storage and VM resources

### Week 3
- VNets, subnets, NSGs, route tables, private/public DNS, peering, and load balancing
- Practice hybrid and internal connectivity concepts
- Review common troubleshooting flows for network access failures

### Week 4
- Azure Monitor, Log Analytics, alerts, backup, recovery, and operational review
- Run through end-to-end labs
- Do timed revision and tighten weak areas

## Repetition Loop
- Read note
- Perform lab
- Write short summary in your own words
- Revisit mistakes 2-3 days later

## What To Memorize
- Resource scope hierarchy
- RBAC concepts and least privilege
- Storage redundancy options and when to use them
- VM sizing, availability choices, and disk behavior
- Core VNet, NSG, routing, and DNS patterns
- Monitoring data sources and alerting concepts

## What To Practice
- Creating and organizing resources
- Assigning roles at the right scope
- Securing storage and network boundaries
- Diagnosing connectivity and deployment issues
- Configuring monitoring and recovery paths
