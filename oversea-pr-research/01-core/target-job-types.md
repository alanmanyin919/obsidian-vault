# Target Job Types

## Purpose
This note is for narrowing down which job types are realistic enough to target in Canada without forcing a premature answer in the personal profile.

## Current Status
- Status: Still exploring
- Rule:
  - Do not lock in target job types until job reality, English burden, and first-year survival value are reviewed together

## Questions To Answer
- Which job types are most realistic given the current background since March 2021?
- Which job types create the best balance between job accessibility and PR usefulness?
- Which job types are realistic despite weaker spoken English?
- Which job types are realistic with a low-chaos landing strategy?

## Candidate Job Types To Investigate
- Application support
- Software developer
- QA tester
- IT support
- Operations support

## Working Rule
- Prefer realism over identity or preference
- Prefer roles that can support survival first, then PR options
