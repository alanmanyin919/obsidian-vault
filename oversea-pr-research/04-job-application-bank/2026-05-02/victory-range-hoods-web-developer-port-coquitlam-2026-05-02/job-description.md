- This is an in-person web-developer role centered on a custom Shopify storefront and related web infrastructure.

- The employer also explicitly expects daily use of AI coding tools such as Codex or Claude Code as part of normal delivery workflows.

- The environment looks closer to an in-house eCommerce and digital-experience team than to a pure SaaS engineering team.

## Main Responsibilities
- Build and maintain a fully customized Shopify theme using Liquid, HTML, CSS or SCSS, and JavaScript
- Create and update custom code sections and storefront functions
- Build new product pages and collection pages aligned with brand standards
- Conduct theme audits and refactoring for code quality and performance
- Monitor website performance and audit for errors daily
- Manage source code through GitHub branching, pull requests, and version control
- Deploy and maintain local-server app hosting environments
- Provision, configure, and monitor Railway-hosted services
- Build and document deployment pipelines from development to production
- Build and maintain Zendesk app integrations, triggers, and webhooks
- Develop and maintain REST and GraphQL API integrations with third-party platforms
- Integrate or audit analytics and tracking tools including Google Tag Manager, GA4, and Meta Pixel
- Use AI tools daily to scaffold, refactor, generate tests and HTML previews, and document code
- Optimize cross-browser and cross-device compatibility
- Improve Core Web Vitals, Lighthouse performance, on-page SEO, and WCAG 2.1 AA accessibility
- Translate design mockups into responsive code
- Write and maintain technical documentation
- Participate in team meetings and provide effort estimates and risk signals

## Stated Requirements

- experience building or maintaining Zendesk app integrations or custom Zendesk apps
- solid understanding of REST APIs and working knowledge of GraphQL
- active daily use of AI coding tools such as Codex, Claude Code, or equivalent
