# Resume Source README

## Purpose
This folder stores the pre-edit resume materials so revision work has a stable baseline.

## What To Put Here
- Your latest original resume before editing
- Older resume versions if they contain useful bullets or project descriptions
- Cover letters only if they contain reusable achievement wording
- Any exported PDF and editable source format you already have

## Naming Suggestion
- `alan-resume-original-YYYY-MM-DD.pdf`
- `alan-resume-original-YYYY-MM-DD.docx`
- `alan-resume-working-copy-YYYY-MM-DD.md`

## Recommended Workflow
1. Keep one untouched original version.
2. Create a working copy for edits.
3. Note major revisions in [[canada-pr-research/07-resume/03-revision-notes/resume-revision-backlog]].

## Notes
- If you want me to revise line by line, the easiest format is plain text or Markdown pasted into a note in this folder.
- If your current resume is only in PDF, we can still review it, but text-based editing will be slower.
